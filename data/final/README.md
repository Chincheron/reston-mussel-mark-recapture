# Final Data

This folder contains cleaned, analysis-ready datasets produced by the pipeline.

## Purpose

* Provide stable inputs for modeling, reporting, and sharing
* Represent the final outputs of each data pipeline

## Characteristics

* Derived from `pipeline/` data
* Fully reproducible
* Ready for downstream use

## Examples

* Modeling datasets
* Aggregated tables
* Export-ready data files

## Rules

* Do not store intermediate results

* Do not manually edit files

* Regenerate from pipeline when needed

* Use clear, descriptive naming

## Notes

Files in this directory should be considered the canonical outputs of the project and suitable for external use.
