# Results

This directory contains all outputs generated from analysis, including figures, tables, and materials prepared for reporting or publication.

## Structure

* `figures/` → visual outputs (plots, charts, maps)
* `tables/` → tabular outputs (summaries, model results)
* `publications/` → finalized materials for sharing (reports, manuscripts, supplements)

## Notes

This directory is intended for outputs that communicate results, not intermediate processing steps. Intermediate or diagnostic outputs should be stored in `data/interim/`.
